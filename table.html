TABLE TAGS

<TABLE> Tags

The <table> Tag in HTML represents tabular data - that is, information presented in a two-dimensional table comprised of rows and columns of cells containing data.

An HTML table is defined with the <table> tag.

Following are some basic tags used for specifying the features of a table:

<tr> tag is used to define each row in a table.
<th> tag is used to define headers of a table.
<td> tag is used to define a table data or a cell.

By default, table headings are bold and centered while data is non-bold and left aligned.
